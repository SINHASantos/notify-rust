pub trait Arg {
    const ARG_TYPE: ArgType;

    fn signature() -> Signature<'static>;

    fn arg_type() -> ArgType { ... }

Types that can represent a D-Bus message argument implement this trait.

Types should also implement either Append or Get to be useful.
